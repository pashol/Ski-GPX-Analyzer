# Changelog

All notable changes to Ski GPX Analyzer are documented in this file.

## [1.2.0] - 2026-01-26

### Added
- **Android Native App Support** - Full-featured native Android application with Capacitor integration
  - Native file picker for GPX/FIT files
  - Hardware-accelerated WebView rendering
  - Intent filters for opening files from other apps
  - Persistent storage via Capacitor Preferences API
  - Network connectivity monitoring
- **Enhanced Map Visualization**
  - Improved fullscreen mode for immersive map viewing
  - Better map controls and responsiveness
  - Refined run cycling feature integration
- **Web Analytics Integration** - Vercel Analytics for usage tracking and performance monitoring
- **Mobile-First Layout Improvements**
  - Responsive design enhancements for touch interfaces
  - Optimized spacing and control positioning for smaller screens
  - Improved run cycle control UI

### Fixed
- Distance unit conversion accuracy in run displays
- Run cycle control spacing issues in bottom navigation
- Fullscreen map mode user experience

### Improved
- Map and run detail view visualizations
- Mobile layout and component spacing
- Overall user experience for both web and native platforms

## [1.1.0] - 2025-12-XX

### Added
- **Internationalization (i18n) Support**
  - Multi-language translation infrastructure
  - Initial language support ready for expansion
- **Ko-fi Integration** - Support button for user donations
- **Run Cycling Feature** - Cycle through runs on map view for easy navigation

### Improved
- Performance and code organization
- Build system with package-lock.json for dependency stability

## [1.0.0] - 2025-11-XX

### Initial Release
- Complete feature set for analyzing ski GPX and FIT files
- Interactive mapping with Leaflet
- Elevation and speed profile charts
- Automatic intelligent run detection
- Performance scoring and statistics
- Session management and persistent storage
- Web and mobile responsive design
